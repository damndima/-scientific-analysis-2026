> **Тема:** Інформаційна система для організації дистанційного навчання з елементами гейміфікації

## 📝 Анотація (Abstract)
Це дослідження охоплює повний цикл розробки та аналізу освітньої ІС. Робота включає систематизацію джерел, професійну верстку в LaTeX, статистичний аналіз даних на Python та побудову прогнозних моделей Machine Learning.

---

## 🚀 Етапи виконання (Лабораторні роботи 1-7)

### 📂 Менеджмент джерел та верстка (ЛР 1-3)
* **Zotero:** Сформовано бібліографічну базу (15+ джерел).
* **LaTeX (Overleaf):** Оформлено статтю за стандартом **IEEE**.
* **Наукометрія:** Налаштовано профілі ORCID та Google Scholar.

### 📊 Аналіз даних та візуалізація (ЛР 4-5)
* **Data Cleaning:** Очищення датасету `Student_Performance.csv` через Pandas.
* **Statistics:** Розрахунок дескриптивних показників та проведення T-test.
* **Visualization:** Створення векторних графіків (Box-plot, Heatmap) для видань рівня **Q1**.

### 🤖 Машинне навчання (ЛР 6)
* Побудовано модель **Linear Regression** для прогнозування `total_score`.
* Оцінка точності: $R^2$ та MSE.
* Проведено **Sensitivity Analysis** (What-if аналіз).
